Bom Management Project - Société Technique Formule SAE (2023-2024)

Project for the Formule SAE team at Polytechnique Montréal. The goal of this project is to create a BOM (Bill of Materials) management system for the team. It includes both a desktop and mobile application to streamline parts tracking and inventory management.
